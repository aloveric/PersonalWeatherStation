# PersonalWeatherStation
Personal Weather Station with Arduino


v 1.0.0

Board and Parts:

- Arduino Uno or Nano
- Bluetooth with CC2541 (VCC, GND, TX, RX)
- Oled 128x64 IIC(I2C) with SSD1306 (VCC, GND, SLC, SDA)
- Passive Buzzer (3.3v, GND, S or D)
- DHT11 (VCC, GND, S or D)
